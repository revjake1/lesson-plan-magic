# Do-Now Patterns

How the skill generates opening tasks that get students working within the
first 60 seconds of class.

### What a Do-Now does

- Activates prior knowledge relevant to today's lesson
- OR spirals back to a concept from earlier this unit
- OR primes a question the lesson will answer
- Runs 3–7 minutes, no teacher-talk start

### Common formats

1. **Spiral review** — one question from earlier this unit
2. **Quick-write / bell-ringer** — open-response prompt tied to lesson objective
3. **Error analysis** — "find and correct the mistake"
4. **Image / primary source response** — one image, one question
5. **Vocabulary activation** — Frayer model or similar for today's key term

### Anti-patterns to avoid

- Do-Nows that require reading long passages
- Do-Nows unrelated to today's learning
- Do-Nows that become the lesson (those are mini-lessons, not warm-ups)

### Output formats

- Single-slide addition to agenda slide
- Printable strip (half-sheet or quarter-sheet)
- Plain-text block the teacher can paste into their LMS manually
