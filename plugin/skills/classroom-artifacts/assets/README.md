# classroom-artifacts / assets

Reusable static assets for the classroom-artifacts skill.

## What to ship in v0.1

- `agenda-slide-template.pptx` — single-slide PPT template matching the
  layout described in `references/agenda-slide-patterns.md`
- `exit-ticket-template-2up.docx` — half-page exit ticket, 2-up per page,
  three variant layouts inside
- `do-now-strip-template.docx` — quarter-sheet strip template
- `sub-plan-template.docx` — full sub plan template matching the pattern in
  `references/sub-plan-patterns.md`

All templates must be:
- Neutral styling (no branding)
- Editable / restylable (use Word or PPT built-in styles)
- License-clean (original work or appropriately attributed public-domain)

## Do not ship

- District-branded templates
- Publisher-proprietary layouts
- Fonts that aren't system-default (Arial, Calibri, Times, Georgia) or
  open-license (Atkinson Hyperlegible, Inter, Open Sans)
